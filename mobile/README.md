# Mobile

This folder contains the mobile app for DentaPass:

- Built with **React Native + Expo**
- Designed for **dental clinic owners** to manage their patient base
- Features include:
  - **Patient management** — view and update patient records
  - **QR code scanning** — quickly look up or enroll patients by scanning their wallet pass QR code
  - **Push notifications** — send targeted notifications to patients directly from the app
