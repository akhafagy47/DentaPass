# Backend

This folder contains the backend for DentaPass:

- **Next.js API routes** — serverless API endpoints powering the DentaPass platform
- **Supabase integration** — database, authentication, and real-time data storage
- **PassKit wallet pass generation** — creates and updates branded Apple Wallet and Google Wallet passes for patients
- **Vercel Cron jobs** — automated recall reminders and review requests sent to patients on a scheduled basis
